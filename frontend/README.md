# zjMall 前端项目

## 技术栈
- Vue 3
- Vite
- Vue Router
- Pinia
- Element Plus
- Axios

## 安装依赖
```bash
npm install
```

## 开发
```bash
npm run dev
```

## 构建
```bash
npm run build
```

## 预览
```bash
npm run preview
```

## API 配置
后端 API 地址配置在 `src/api/config.js` 中，默认指向 `http://localhost:8080/api/v1`

## 功能
- ✅ 用户注册/登录
- ✅ 商品列表/搜索
- ✅ 商品详情
- ✅ 个人中心
- ⏳ 购物车（开发中）
- ⏳ 订单管理（开发中）
