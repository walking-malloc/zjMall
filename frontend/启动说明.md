# 前端项目启动说明

## 前置条件
1. 确保后端服务已启动（网关服务运行在 http://localhost:8080）
2. 确保已安装 Node.js（版本 >= 16）

## 启动步骤

### 1. 安装依赖（如果还没安装）
```bash
cd frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问前端
浏览器打开：http://localhost:3000

## 功能说明

### 已实现的功能
- ✅ 用户注册（需要手机号和验证码）
- ✅ 用户登录（手机号+密码）
- ✅ 首页展示（热门商品）
- ✅ 商品列表（支持搜索和分页）
- ✅ 商品详情页
- ✅ 个人中心（显示用户信息）

### 待开发功能
- ⏳ 购物车
- ⏳ 订单管理
- ⏳ 地址管理

## 注意事项

1. **API 地址配置**：在 `src/api/config.js` 中配置，默认指向 `http://localhost:8080/api/v1`
2. **跨域问题**：Vite 已配置代理，开发环境会自动代理 `/api` 请求到后端
3. **Token 存储**：登录后 token 存储在 localStorage 中
4. **用户信息**：登录/注册成功后，用户信息会保存在 Pinia store 中

## 常见问题

### 1. 无法连接后端
- 检查后端服务是否启动
- 检查 `configs/config.yaml` 中的网关端口是否为 8080
- 检查浏览器控制台是否有 CORS 错误

### 2. 登录后无法获取用户信息
- 检查 token 是否正确保存
- 检查后端登录接口是否返回用户信息
- 查看浏览器控制台的网络请求

### 3. 商品列表为空
- 检查数据库中是否有商品数据
- 检查商品服务是否正常运行
- 查看浏览器控制台的错误信息

## 开发建议

1. 使用浏览器开发者工具查看网络请求和响应
2. 使用 Vue DevTools 查看 Pinia store 状态
3. 查看浏览器控制台的错误信息进行调试
